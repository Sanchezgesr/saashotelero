## 🛠️ Stack Tecnológico

```
Frontend     → Next.js + React + TypeScript + Tailwind CSS
Backend      → Supabase (PostgreSQL + Auth + RLS + Storage)
Hosting FE   → Vercel
Hosting BE   → Supabase (cloud)
Control      → GitHub
```

## 🎯 Objetivo del Producto

Reemplazar cuadernos y hojas de cálculo en hospedajes de Junín con un sistema web moderno, accesible desde celular, con gestión de habitaciones, reservas, caja y reportes.
